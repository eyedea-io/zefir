## Manage document head

```js
import Head from 'zefir/head'

const Landing = () => (
  <div>
    <Head>
      <title>My awesome page title</title>
      <link href='https://fonts.googleapis.com/css?family=Roboto:300,400,500,700' rel='stylesheet' />
    </Head>
  </div>
)

export default Landing
```
